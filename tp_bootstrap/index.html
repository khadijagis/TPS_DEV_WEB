Notre équipe :

Tim Berners-lee

Tim Berners-Lee, un informaticien britannique, a inventé le HTML pour permettre la création de pages Web lisibles sur Internet. Il est aussi à l'origine du World Wide Web (WWW).

Håkon Wium Lie

Håkon Wium Lie, un ingénieur norvégien, a proposé les feuilles de style en cascade (CSS) pour séparer le contenu des sites Web (HTML) de leur présentation visuelle (design).

Brendan Eich

Brendan Eich a développé JavaScript en seulement 10 jours alors qu'il travaillait chez Netscape Communications. Il a été conçu pour ajouter de l'interactivité aux pages Web.
